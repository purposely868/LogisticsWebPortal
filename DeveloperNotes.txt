Nodejs Express.
    npm init in NodeServer regular initialization. Creating a package.json
    npm install express --save (save it as dependency for production.)
    BUT we will use the Express application generator to generate a skeleton express app

Express App Generator:
    1. npx express-generator -v ejs --git => Express application generator (Node.js 8.2.0) -v:viewing engine with ejs + --git: for generating git ignore file. (.gitignore has pre defined files/folders. Our main ignore is node_modules)
    2. npm install
    3. writing in package.json a new script. "devstart": "SET DEBUG=nodeserver:* & nodemon"
    4. npm run devstart

If Copying from repository the package.json will have all the dependencies for development and production.
